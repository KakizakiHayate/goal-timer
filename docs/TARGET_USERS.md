# ターゲットユーザー定義

このドキュメントは、Goal Timerアプリのターゲットユーザーを定義します。
Issue作成時や機能提案時に、このドキュメントを参照してユーザーに適切な提案を行ってください。

## プライマリターゲット

### 小学校高学年〜高校生（10〜18歳）

| 項目 | 内容 |
|------|------|
| **年齢層** | 10〜18歳 |
| **目的** | 中学受験、高校受験、大学受験、定期試験対策 |
| **特徴** | 学校終わりに勉強する習慣をつけたい、学習時間を可視化してモチベーション維持したい |
| **課題** | 勉強を続けられない、スマホやゲームに気が散る |

**ペルソナ例**
- 中学受験を目指す小学5年生。塾の宿題や自主学習の時間を記録して、毎日の頑張りを可視化したい
- 高校受験を控えた中学3年生。学校終わりに自宅で勉強する習慣をつけたい。科目ごとの学習時間を管理したい
- 大学受験を控えた高校2年生。毎日の勉強時間を記録して、目標達成までの進捗を確認したい

## セカンダリターゲット

### 大学生（19〜25歳）

| 項目 | 内容 |
|------|------|
| **年齢層** | 19〜25歳 |
| **目的** | 資格試験、大学の試験対策、就活準備 |
| **特徴** | 自己管理が必要な環境で、学習時間を効率的に管理したい |
| **課題** | 自由時間が増えて自律が難しい、SNSやYouTubeに時間を奪われる |

**ペルソナ例**
- TOEIC 800点を目指す大学3年生。科目ごとの学習時間バランスを把握したい
- 公務員試験を控えた大学4年生。毎日の勉強時間を記録して計画的に学習を進めたい

## ターゲット共通の特徴

### 求めていること
- シンプルで使いやすいUI
- 学習時間の可視化・記録
- 継続のモチベーション維持
- 目標達成までの進捗把握

### 避けたいこと
- 複雑な機能・設定
- 広告による中断
- SNS的な要素（他人との比較）

---

## 競合との差別化ポイント

Goal Timerが提供する独自価値：

| 機能 | 説明 |
|------|------|
| **ネガティブ回避設計** | 「これをやらないとどうなるか？」を明文化し、行動の動機を引き出す |
| **目標数の制限** | 無制限に目標を作れず、「本当に重要なこと」だけに集中できる |
| **タイマー3モード** | 通常・カウントダウン・ポモドーロの3方式で目的に応じて選べる |
| **シンプルUI** | 男性・女性問わず"努力する自分"が恥ずかしくない中性的デザイン |

## Issue作成時の参照ガイド

新機能やUIの提案時は、以下を確認してください：

1. **プライマリターゲット（10〜18歳の小中高生）に価値があるか？**
   - 学習時間管理に役立つか
   - シンプルで直感的か（特に小中学生でも迷わず使えるか）

2. **ターゲットの課題を解決するか？**
   - 継続の支援になるか
   - モチベーション維持に貢献するか

3. **競合との差別化になるか？**
   - 参考モデル「目標達成タイマー」との違いは何か
   - Goal Timer独自の価値を提供できるか

4. **避けるべき機能**
   - 複雑すぎる設定（小中学生が理解できない機能）
   - SNS的な機能（ランキング、シェア強制など）
   - ターゲット層に合わないゲーミフィケーション
